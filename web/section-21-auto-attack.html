Auto attack

Use Auto Attack: Never for Tinker and Earth Spirit

Use Auto Attack: Standard for everything else.

You definitely want to have Summoned Unit Auto Attack: Always for manta and illusion based heroes such as Phantom Lancer, Terrorblade and Naga Siren.

This works: Auto Attack: Standard + Summoned Unit Auto Attack: Always

Another option is to use Toggle Autoattack button.

This allow you to use Summoned Unit Auto Attack: Same as Hero and switch the state with the button.

However, I personally don't like Autoattack: On on my main hero.
I use Auto Attack: Standard + Summoned Unit Auto Attack: Always.

Experiment with it.

And don't forget to enable Quick Attack and Quick Move.
