Directional Move

Sometimes you want to turn your hero towards a direction exactly.

That's why Directional Move is used all the time on heroes like Shadow Fiend, Slark, Ursa, Earth Spirit.

Because it's much easier to line up your spells. So, There are 2 ways to bind it.

CapsLock is a great key for Directional Move, because you activate it with the pinky finger.
This allows you to use all the abilities and items as usual.

  • Directional Move: CapsLock

With the August 19, 2025 update Valve added this:

Holding Alt Enabled Direcional Move

However, I do not recommend using this.
Because, while yes it's more convenient for Directional Move it really makes everything else worse.

Think about it, we use Alt for everything:
- Self cast abilities, items - Toggle abilities, items - All things shop and courier - Pings on things

We optimized our hotkey layout around Alt.
This setting gets in the way and makes the experience worse. Use a normal hotkey instead like CapsLock or Win

p.s. this setting has always been in the game, but you had to set it up in console or autoexec.cfg like this:

  • bind "ALT" "+dota_directional_move"
  • cl_dota_alt_unit_movetodirection true
