Select Unit (advanced)

Here's another way to select and control units in Dota 2.

Dota 2 has these fantastic features:
Select Hero, Select All Other Units, Select All Units and Unified Orders.

1. Select Hero — self-explanatory.

2. Select All Other Units selects all units under your control except your hero.

3. Select All Units selects all units under your control.
In other words, Select Hero + Select All Other Units.

4. Unified Orders is a modifier key — Ctrl by default.

Unified Orders allows us to issue commands to all units at once without selecting them.
This is very convenient if used occasionally, but it's really annoying to hold Ctrl all the time if we need to control all units all the time.

These options should supplement your control groups — not replace them completely.

In my opinion there are 3 ways to approach these keybinds:

Option A) - Bind these as ALT Control Group
Option B) - Bind these in place of some Control Groups
Option C) - Bind these to the mouse wheel

There are advantages and disadvantages to each.

I consider the entire layout as a whole, simplicity, key 'priority' and how general it is for most players in most situations playing all heroes.

Without writing a massive essay about this, in short, I do recommend the Option C.

  • Select Hero: WheelUp
  • Select All Other Units: WheelDown
  • Select All Units: WheelDown + Alt
  • Unified Orders: CTRL

Select Hero and Select All Other Units are the "opposite polarities".

Select All Units is Select All Other Units + Alt, which is "to me" modifier.

Do not bind WheelUp + Alt to anything.
You want to Select Hero with WheelUp and WheelUp + Alt

Now, these are somewhat weird binds.

But because we manage our units with Control Groups we are not going to use these often.

I encourage you — the reader — to go into a Demo and play around with Naga Siren or Terrorblade.
