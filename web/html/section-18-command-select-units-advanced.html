Select Units (advanced)

There is another way to select and control units in Dota 2.

Dota 2 has these fantastic features:
Select Hero
Select All Other Units
Select All Units
Unified Orders

1. Select Hero is self-explanatory.
The good thing — you don't need it.

Just assign your hero to 1Control Group 1 — this gives you the same effect, but provides additional flexibility of a control group.

Example:
Naga Siren players usually run the main hero + one illusion together.

2. Select All Other Units selects all units under your control except your hero.

3. Select All Units selects all units under your control.
In other words, Select Hero + Select All Other Units.

4. Unified Orders is a modifier key — Ctrl by default.
Unified Orders allows us to command all units at once without selecting them.

This is very convenient when used occasionally.

These options should supplement your control groups — not replace them completely.

In my opinion there are 3 ways to approach these keybinds:

Option A) - Bind these as ALT Control Groups
Option B) - Bind these in place of some Control Groups
Option C) - Bind these to some other keys

There are advantages and disadvantages to each.

Let's consider Option A:

  • Select Hero: 1   (set to Control Group 1)
  • Select All Other Units: Alt   +   1
  • Select All Units: Alt   +   2

This makes a lot of sense.

The first downside is that we already use Alt1 for Open Shop and Alt2 for Quickbuy

So this breaks the 'elegancy' of handling all-things-shop with Alt 12345.

The second downside is that Alt + key might be not very fast (or easy) to use as a 'Select' key, especially if you plan to use these often.

Let's consider Option B:

  • Select Hero: 1   (set to Control Group 1)
  • Select All Other Units: 2
  • Select All Units: 3

The main advantage here is that it's the fastest way to use these actions.

The downside is that you lose access to the control groups 2 and 3.
So this is worse when you actually want to control individual units or a specific group of units.

To be fair, you can still bind units individually to 4 5 and beyond.
And because our Attack is bound to Spacebar it might not a big problem.

Another downside of this option is that you lose continuity with other games in which 123 are the 'normal' control groups.

Let's consider Option C:

If you're willing to nudge away F1Select Courier or `Upgrade Talent — you can actually do it.

  • Select Hero: 1   (set to Control Group 1)
  • Select All Other Units: `
  • Select All Units: F1

Again, it depends on your individual playstyle and priorities.

Option D?

  • Select Hero: WheelUp
  • Select All Other Units: WheelDown
  • Select All Units: WheelDown   +   Alt

Now, these are weird binds.
But this is what I'm currently using.

I manage all my unit control with Control Groups — I barely use these commands.

Fun Fact!

Naga Siren has illusions, but Terrorblade has "images".

The difference?

You cannot assign individual images to different control groups.
When you assign an image to a control group that control group will always recall all images.

Not sure if intended.
