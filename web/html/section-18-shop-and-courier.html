Shop & Courier

There are many different configurations for all-things-shop.
This is what I'm currently using.

  • Select Courier: F1
  • Open Shop: Alt+1
  • Quickbuy: Alt+2
  • Courier Deliver: Alt+3
  • Courier Burst: Alt+4
  • Courier Shield: Alt+5
  • Buy Sticky: Y

In any case, you want the following pattern to be easy:
QuickBuy → Courier Deliver → Courier Boost

F1Select Courier is right next to our control groups.
This makes sense, because it's "Select Something" key.

YBuy Sticky is not used very often, but I prefer having hotkeys for everything.
This action used to be much more useful, before we weren't getting free Teleport Scrolls on deaths.
