Abilities - No Target Abilities

The last extremely nitpicky thing on Ability binds.

No Target abilities are these:

Mirror Image, Rip Tide, Song of the Siren, Quas, Wex, Exort, Invoke, ..., etc

If a 'No Target' ability is bound to Q, then AltQ will not fire it.
The input is 'consumed' with no effect.

And because we're going to use ALT often, we don't want that to happen.

So, bind these to Normal Cast and Quick Cast

Normal Cast — Q
Quick Cast — AltQ

This might be confusing, but all we're doing here is making sure that both Q and AltQ point to Mirror Image, so it triggers on Q with or without the Alt key.

Same with Song of the Siren, with R and AltR respectively.
