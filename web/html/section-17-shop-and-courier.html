Shop & Courier

Shop Settings

Uncheck these:
Shop Always Uses Hotkeys
Shop Search Gets Focus On Open
Enable Simple Shop

Shop Search Gets Focus On Open makes whatever you type go into the search bar of the Shop.

Well, that makes sense...
HOWEVER, it consumes all the keys.

Even keys (and key combinations) like F1 and AltF1 which can't even be typed if you think about it.

This results in a scenario you don't want:

You open the shop and suddenly you need to react to something.

You can't cast or select anything until you close the shop because all the buttons you press will attempt to go into the search bar of the shop.
Closing the shop takes time, so it's very likely to get you killed as you can't react immediately.

I prefer my hotkeys working all the time, thank you, and for that reason I'm out.

Shop Hotkeys

There's definitely a balance between simplicity and optimization.
Here's the simplest way to go about all-things-shop:

  • Select Courier: F1

  • Open Shop: Alt   +   1
  • Quickbuy: Alt  +   2
  • Courier Deliver: Alt  +   3
  • Courier Burst: Alt  +   4
  • Buy Sticky: Alt  +   5

F1Select Courier is "select" key.
So having it close to our Control Groups makes sense.

You want the following pattern:
QuickBuy → Courier Deliver → Courier Burst

And it doesn't get any simpler than ALT2 → 3 → 4

Buy Sticky

Buy Sticky used to be much more useful before we weren't getting free Teleport Scrolls on deaths.
This was introduced in the 7.29 update (along with Dawnbreaker and Water Runes, and yes — you are getting old)

It's still useful though.
Especially for core players, because they die less than they teleport around.

Long story short, bind Buy Sticky to ALT5 as a "low priority extra":

  • Open Shop: Alt   +   1
  • Quickbuy: Alt  +   2
  • Courier Deliver: Alt  +   3
  • Courier Burst: Alt  +   4
  • Buy Sticky: Alt  +   5

Consider this...

Interestingly, I used ALTMouseWheelUp for Open Shop for about a year.
And I honestly liked it.

Despite how strange it sounds, it plays very well, because you 'scroll up to see a different layer'.
I promise it's not bad.

But the main benefit is that it allows you to shift the whole thing 1 key to the left:

  • Open Shop: Alt   +   MouseWheelUp
  • ← Quickbuy: Alt  +   1
  • ← Courier Deliver: Alt  +   2
  • ← Courier Burst: Alt  +   3
  • ← Buy Sticky: Alt  +   4

This is a killer layout because it solves every problem we faced in this section before I deleted a massive wall of text.

A possible downside is that you lose continuity with other games that might not allow you to bind AltMouseWheelUp as your shop / inventory.

But it's probably the best for speed, because it allows you to have instant access to everything.
